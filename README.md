# 🚀 LinguaMaster – Language Learning App

A modern and interactive Language Learning Application developed using **Flutter** and **Firebase** as part of **Optimus Automate Internship – Task 2**.

LinguaMaster helps users improve their language skills through vocabulary lessons, flashcards, quizzes, pronunciation support, progress tracking, and achievement badges in a user-friendly environment.

---

# 📌 Project Overview

LinguaMaster is designed to make language learning simple, engaging, and effective.

The application allows users to:

* Learn new vocabulary words
* View Urdu translations
* Listen to word pronunciations
* Practice with interactive flashcards
* Take quizzes to test knowledge
* Track learning progress
* Unlock achievements
* Discover a daily word
* Manage personal profiles securely
